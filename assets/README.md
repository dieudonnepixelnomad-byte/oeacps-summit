# Assets

Place your images and icons here.

## Images
Directory: `assets/images/`

Required files for full experience:
- `hero_placeholder.png` (Used in Home, Info)
- `avatar_1.png` to `avatar_5.png` (Speakers)
- `news_1.jpg`, `news_2.jpg`, etc. (News)
- `media_1.jpg`, `media_2.jpg` (Media gallery)

If files are missing, the app will show a grey placeholder.
